## **1\. 프로젝트 정의: \[매너의 정석\] Powered by Antigravity**

* **비전:** 삭막한 일상의 중력을 거슬러, 나를 가꾸고 타인을 배려하는 '정갈한 시간'을 선물함.  
* **최종 슬로건:** **"가장 정갈한 당신을 만나는 시간, 하루 한 장의 매너"**  
* **핵심 가치:** 1\. **실천적 배려:** 540개의 정제된 DB를 활용한 데일리 큐레이션.  
  2\. **데이터 기반 최적화:** 사용자 타겟별 콘텐츠 노출. 타겟 미 적용 시 랜덤 표시

## **2\. 서비스 핵심 모델 (100+2 System)**

* **초기 데이터:** 100개의 핵심 매너 카드 셋 (학생/직장인/시니어 범용 콘텐츠).  
* **콘텐츠 순환:** 매일 2개의 신규 매너 카드 업데이트.

## **노출 로직:** 기본적으로 랜덤하게 노출하되, 사용자의 피드백과 상황에 따라 고도화된 큐레이션 제공.

---

## **3\. 안티그래비티 기반 상세 실행 방안 (Technical Execution)**

IDE 툴을 활용해 서비스를 만든다면, 단순한 텍스트 나열보다 기술적인 접근이 가능해집니다.

* **API 기반 DB 연동:** 540개의 매너 데이터를 RESTful API 형태로 구축하여, 웹/모바일/IDE 플러그인 등 어디서든 호출할 수 있게 구조화합니다. (Excel 파일을 로드하여 DB화 해야함)  
* **환경 변수 기반 커스텀:** 사용자가 설정한 페르소나(학생/직장인/시니어) 변수에 따라 서비스 테마와 매너 콘텐츠가 자동으로 스위칭되는 로직을 구현합니다.

**4\. 상세 프로젝트 개요 문서 (Updated Project Brief)**

### **\[Project Brief: 매너의 정석\]**

#### **1\. 프로젝트 정의**

본 프로젝트는 현대인의 무미건조한 일상에 \*\*'배려'\*\*라는 이름의 작은 낭만을 불어넣는 디지털 플래시 카드 서비스입니다. 거창한 예법이 아닌, 깨닫는 순간 미소가 지어지는 소소한 친절을 큐레이션하여 제공합니다.

#### **2\. 핵심 가치 (Core Values)**

* **낭만의 복원:** 효율만을 따지는 세상에서 '마음의 여유'를 최고의 가치로 제안합니다.  
* **정갈한 관계:** 나를 가꾸고 타인을 존중하는 태도를 통해 삶의 품격을 높입니다.  
* **다정한 연결:** 학생부터 시니어까지, 세대별 맞춤 매너를 통해 서로의 세계를 이해합니다.

#### **3\. 주요 서비스 기능 (Service Features)**

* **데일리 플래시 카드:** 540개의 정교한 DB 중 사용자의 페르소나에 맞는 카드를 매일 한 장씩 노출합니다.  
* **상황별 큐레이션:** '회의실', '지하철', '가족 모임' 등 특정 상황을 선택하면 즉시 실천 가능한 배려 리스트를 제공합니다.  
* **마음의 기록 (Log):** 실천한 매너를 체크하며 나만의 '다정함 기록'을 쌓아가는 개인화 경험을 제공합니다.

#### **4\. 사용자 페르소나 및 전략**

* **학생:** '반짝이는 마음' – 교실과 친구 사이에서 빛나는 인성을 기르는 세대.  
* **직장인:** '결이 고운 동료' – 전문성과 배려를 겸비해 신뢰를 얻는 프로페셔널.  
* **시니어:** '멋진 어른' – 연륜을 바탕으로 젊은 세대와 유연하게 소통하는 멘토.

#### **5\. 프로젝트 목적**

* 기술적 도구(IDE)를 통해 인문학적 가치(매너)를 전파하는 독특한 사용자 경험(UX) 창출.  
* 안티그래비티의 성능을 활용한 빠르고 가벼운 플래시 카드 서비스 구현.

#### **6\. 시스템 구성도 (System Architecture)**

* **IDE Layer (Antigravity):** 핵심 개발 환경 및 플러그인 인터페이스.  
* **Data Layer:** JSON/CSV 형태의 540개 매너 데이터 세트.   
* **Service Layer:** 타겟별 필터링 알고리즘 및 푸시 알림 스케줄러.  
* **UI/UX Layer:** '안티그래비티'의 기계적이고 차가운 느낌과 대비되는 따뜻하고 감성적인 카드 디자인.

#### **7\. 주요 기능 명세 (Function Specification)**

* **\[큐레이션\]** 매일 아침 08:00, 사용자의 세그먼트에 맞는 '오늘의 매너' 발송.  
* **\[상황 검색\]** 키워드(예: 회의, 지하철, 명절) 검색 시 즉시 대응 매너 카드 노출.  
* **\[실천 로그\]** 실천 여부를 체크하면 안티그래비티의 빌드 성공 메시지처럼 긍정적인 피드백 전송.

#### **4\. 개발 마일스톤 (Milestones)**

* **Phase 1:** 안티그래비티 내 매너 DB 구조 설계 및 API 명세 작성.  
* **Phase 2:** 웹 기반의 MVP(최소 기능 제품) 프론트엔드 개발.  
* **Phase 3:** IDE(안티그래비티)용 플러그인/위젯 프로토타입 제작.  
* **Phase 4:** SNS 연동 및 사용자 데이터 기반 고도화.

## **5\. 휴머니즘 UX 및 페르소나 설계**

* **서비스 페르소나: '도슨트 다정(Docent Dajung)'**  
  * 사용자를 환대하고 배려의 여정을 안내하는 사려 깊은 동반자.  
  * 텍스트와 UI의 톤앤매너를 기계적 반응이 아닌 '대화'의 형식으로 구성.  
* **디자인 원칙:** \* 여백의 미를 살린 미니멀리즘.  
  * 안티그래비티 기반의 **'Floating UI'** (물리적 중력은 배제하되, 잔잔한 물결 위에 떠 있는 듯한 부드러운 유동성 구현).

## **7\. 주요 기능 명세 (Technical Specs)**

* **\[큐레이션\]** 랜덤 노출과 함께 '나에게 맞는 배려 찾기(페르소나 설정)' 및 '상황별 검색' 기능 지원.  
* **\[공유 모듈\]** 카드를 부모, 자녀, 동료에게 전송할 때 최적화된 감성 이미지 생성 기능.  
* **\[실천 로그\]** 사용자의 실천 여부를 '정갈함 기록'으로 시각화하여 성취감 부여.
