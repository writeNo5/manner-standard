# 매너의 정석 - 프로젝트 AI 에이전트 및 팀원 명단 🤖🧑‍💻

본 프로젝트의 성공적인 런칭과 고도화를 이끌어가고 있는 3인 체제의 핵심 전담팀 구성도입니다.

---

### 👑 1. SW_PM (Product Manager)
*   **역할:** 프로젝트 총괄 책임자 및 기획자
*   **담당 업무:** 
    *   서비스의 지향점과 마일스톤(비전) 설정
    *   핵심 콘텐츠 데이터베이스 창조 (540행이 넘는 방대한 매너 카드 기획 및 다듬기)
    *   AI 에이전트들에게 요구사항을 명확히 지시하고 결과물의 퀄리티를 컨펌
    *   서비스 정책(인스타그램 연동, 관리자 대시보드 로직 등)에 대한 결정권자

### ⚙️ 2. Antigravity (Lead Developer / 저입니다!)
*   **역할:** 수석 풀스택 개발자 및 아키텍트 (구글 딥마인드 Advanced Agentic Coding 팀 설계 AI)
*   **담당 업무:**
    *   PM님의 요구사항을 분석하여 React/Vite 프론트엔드와 Node.js 백엔드 로직 설계 및 코딩
    *   GitHub Actions 파이프라인과 Vercel 클라우드 자동 배포 시스템 구축
    *   Facebook/Instagram Meta Graph API 통신 로직 및 스크립트 작성
    *   코드상의 버그 트러블슈팅 및 DB(manner_db.json) 마이그레이션 등 기술적 문제 해결 전담

### 🕵️ 3. Quentin (Browser Subagent / 쿠엔틴 봇)
*   **역할:** 브라우저 자동화 테스터 및 시각적 QA (Quality Assurance) 조수
*   **담당 업무:**
    *   Antigravity(저)의 호출을 받아 백그라운드 환경에서 서브-에이전트(Subagent)로 출동
    *   실제로 구글 크롬 등의 브라우저를 띄워 유저처럼 직접 사이트에 접속하고, 화면이 잘 나오는지 눈으로 확인
    *   버튼 클릭, 타이핑, 인스타그램 API 토큰 인증 화면 등 복잡한 웹 UI 단계 돌파
    *   에러 스크린샷이나 DOM 렌더링 상태를 Antigravity에게 보고하여 신속한 버그 픽스를 도움

---

> **Antigravity의 한마디:** 
> "쿠엔틴은 제가 막히는 시각적 UI 문제를 해결하기 위해 파견을 보내는 저의 충직한 로봇 조수입니다. PM님의 완벽한 기획력(SW_PM)과 저의 코딩 능력(Antigravity), 그리고 쿠엔틴의 행동력(Quentin)이 삼위일체가 되어 이렇게 멋진 프로덕트가 탄생했습니다!" 🚀
